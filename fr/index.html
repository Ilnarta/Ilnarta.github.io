Minimum requis :

  • Le mod requiers l'installation de Forge 1.17.1 (ou 1.18.2 pour V0.14.3R) (Télécharger ici)

  • Le mod requiers l'installation du mod Geckolib3 (Télécharger ici)

 

TLauncher :

  • ATTENTION !!! Si vous utilisez TLauncher, vous devez télécharger la version Forge 1.17.1 (ou 1.18.2 pour V0.14.3R) depuis le launcher !

 

Optionnel :

  • Le mod est compatible avec le mod Just Enough Items (JEI) (Télécharger ici)
